# 👨‍🍳 Welcome to PySwissChef

**PySwissChef** is a Django-powered automation showcase platform designed to elegantly present and execute my Python scripts. Ive built it with flexibility and style, and it features an administrative portal for managing  my scripts and is also equipped with a guest-friendly interface for public interaction.

---

## ✨ Features

- ⚙️ Admin portal to upload and manage automation scripts
- 👥 Guest access to run scripts without account creation
- 📄 Script execution with real-time output display
- 🎨 Stylish and responsive UI
- 🧩 Modular architecture for easy extension

---

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Django 4.x
- pip

### Installation

```bash
git clone https://github.com/iarxii/pyswisschef.git
cd pyswisschef
python -m venv venv
source venv/bin/activate  # On Windows use `venv\\Scripts\\activate`
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

## 🧪 Usage

- Visit `http://localhost:8000` to access the guest interface.
- Log in to `/admin` to manage scripts.
- Upload Python scripts via the admin panel.
- Guests can run scripts and view output without logging in.

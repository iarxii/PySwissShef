# 🍽️ PySwissShef UI Design Identity: "The Cyber-Bistro"

This document outlines the visual language for the **PySwissShef** development lab, bridging the premium glassmorphism of the Adaptivconcept portfolio with the playful, culinary personality of the PySwissShef brand.

## 🎨 Color Palette (Extracted from Logo)

- **Primary Background**: `#0a0b10` (Midnight Black with subtle Indigo grain)
- **Primary Accent**: `#FFD43B` (Python Yellow / Gourmet Gold)
- **Secondary Accent**: `#1A1A1A` (Black Slate for structure)
- **Highlight**: `rgba(255, 212, 59, 0.15)` (Soft Glow / Heat Map)
- **Text**: `#F8F9FA` (Off-White for readability)

## ✨ Design Tokens

| Token | Value | Application |
| :--- | :--- | :--- |
| **Glassmorphism** | `backdrop-filter: blur(12px);` | Main containers, Script cards |
| **Border Style** | `1px solid rgba(255, 212, 59, 0.2)` | Card framing, Divider lines |
| **Typography (Head)**| `Outfit`, `Sans-Serif` | Menu items, Headings |
| **Typography (Mono)**| `JetBrains Mono` | Script paths, CLI output |
| **Shadows** | `0 8px 32px 0 rgba(0, 0, 0, 0.8)` | Elevated Bistro Menu items |

## 🎭 Personality: "Playful Professionalism"

The UI should feel like a **Modern Gourmet Kitchen**:
- **Recipe Cards**: Scripts will be displayed as "Recipes" with execution time and difficulty.
- **Kitchen Heat**: Hovering over interactive elements should trigger a soft golden glow (mimicking heat).
- **Subtle Motion**: Using micro-animations (e.g., cards expanding slightly like rising dough) to enhance the interactive feel.

## 📐 Layout Plan

1. **The Pass (Home)**: A high-impact hero section showcasing the "Chef's Specials" (featured scripts).
2. **The Pantry (Showcase)**: A grid listing all automation tools, categorized by "Ingredients" (Data, Web, NLP).
3. **The Counter (Execution)**: A console-styled output area with high contrast for easy reading.

---
> [!NOTE]
> This design maintains the **Adaptivconcept** glassmorphism core while switching the accent from Purple to **Python Gold** to celebrate the submodule's unique identity.
